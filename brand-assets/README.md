## SVG Standards

All files follow these conventions:
- `xmlns="http://www.w3.org/2000/svg"` on root element
- Explicit `width` and `height` attributes
- `viewBox` for scalable rendering
- Inline styles only (no CSS classes)
- `<defs>` with glow filter where applicable
- Fully self-contained (no external dependencies)
- Transparent background for logo files
- Dark `#060C06` background for social/token context files

## Typography

- Display: Sora (700/800 weight)
- Body: Inter
- Mono: JetBrains Mono
- All wordmarks use `letter-spacing` for the tracked-out aesthetic

## The Convergence Prism

The mark is composed of 3 triangular polygon faces forming a faceted gem/prism shape:
- **Left face**: Primary brand color at 0.9 opacity
- **Right face**: Lighter variant at 0.75 opacity
- **Bottom face**: Darker variant at 0.85 opacity

Base coordinates at `viewBox="0 0 100 100"`:
```
Top vertex:    (50, 8)
Left vertex:   (20, 44)
Right vertex:  (80, 44)
Center vertex: (50, 56)
Bottom vertex: (50, 92)
```
